多容器通信

学习目标

项目往往都不是独立运行的，需要数据库、缓存这些东西配合运作。
这节我们把前面的 Web 项目增加一个 Redis 依赖，多跑一个 Redis 容器，演示如何多容器之间的通信。

本文档课件配套 视频教程

更多相关命令

docker stop/start container-id 停止/启动指定 id 的容器
docker rmi image-id 删除指定 id 的镜像

docker network ls 查看网络列表
